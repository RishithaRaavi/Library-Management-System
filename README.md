# Library Management System
A simple Library Management System built using Python and SQLite.
## Features
- Admin Login
- Add Book
- View Books
- Search Book
- Update Book
- Delete Book
- Add Member
- View Members
- Search Member
- Issue Book
- Return Book
- Reports
## Technologies Used
- Python
- SQLite
- VS Code
## How to Run
1. Clone the repository.
2. Run:
```bash
python database.py
python main.py
```
